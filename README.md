<h2> 🏗️ ### README.md Content:
coach-alan.github.io | The Architect's Hangar
This repository serves as the central hub and production pipeline for Coach Alan's instructional design frameworks. It is the home of the Clinical AI Residency—a technical ecosystem engineered to bridge the gap between Federal Physical Activity Guidelines and Generative AI.


🍱 Project Architecture
This site utilizes a Bento Box design philosophy, delivering modular, high-fidelity instructional components that are:

Mobile-First: Optimized for 320px width (Canvas LMS/Mobile Browsers).

WCAG 2.1 AA Compliant: Accessible, semantic HTML for screen-reader compatibility.

SSoT-Grounded: All logic is anchored to the PAGA 2018 (2nd Ed) and ACSM 12th Ed.

🛠️ The Hardened Pipeline
We utilize Infrastructure-as-Code principles to ensure instructional agility via a dual-silicon, AI-native stack:

Pilot Command: Antigravity IDE (M1 MacBook Pro) for architectural development and version control.

Clinical Engine: Gemma 4 (M2 MacBook Air) for high-fidelity clinical logic, math verification, and auditing.

Deployment: GitHub Pages (Jekyll) for web-based field guides and direct Canvas LMS injection.

🔄 The Methodology: The Mission Loop
All content and student workflows follow a strict clinical auditing framework:

Pattern: Identifying physical inactivity or clinical markers.

Rule: Applying the federal standards of the PAGA 2018.

Solve: Engineering evidence-based interventions and prescriptions.

🔗 Active Work Products
Instructor Field Guide: The 15-Pillar Vault for AI-augmented Kinesiology.

The Centaur Protocol: Our master logic for Human-in-the-Loop AI integration.
